# Учебный проект Яндекс.Практикума "Место"

## Интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.

### Использованные технологии
*HTML
*CSS
*Grid Layout / Адаптивная верстка
*Flex
*Javascript
*Webpack
